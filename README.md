## Descripción

Este repositorio contiene la documentación oficial del proyecto **Ahorra YA**, una aplicación web orientada a la consulta de tiendas, ofertas, favoritos, reseñas y funcionalidades geográficas mediante un mapa interactivo.

La documentación fue elaborada con **MkDocs**, una herramienta FOSS para generar sitios estáticos de documentación a partir de archivos Markdown.

El objetivo de este repositorio es centralizar la información necesaria para comprender, instalar, configurar, usar y mantener el proyecto.

---

## Contenido de la documentación

La documentación incluye:

- Manual de usuario.
- Documentación técnica.
- Arquitectura general del sistema.
- Descripción del backend.
- Descripción del frontend.
- Integración Backend-Frontend.
- Funcionalidades SIG.
- Base de datos y diccionario de datos.
- Instalación y configuración.
- Versiones y dependencias.
- Referencia de API.
